# Sea simulator
This is a simple simulation of a sea with fish and sharks.

## Requirements 
- MacOS aarch64 
- C++17
- CMake 3.28 

## Installation 

```bash 
$ git clone https://github.com/qvanle/sea.git
$ cd sea 
$ brew install glfw
$ brew install cmake
$ ./build.sh
```
